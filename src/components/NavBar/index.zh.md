## `RightIcon`

| 属性名 | 描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| icon | Icon图标组件 | `Icon` | - |
| onPress | 图标点击事件 | `function` | - |
| label | 图标功能说明文本 | `string` | - |


## `Icon`

| 属性名 | 描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| color | Icon颜色 | `string` | - |
| size | Icon尺寸 | `string` | - |
| fillColor | Icon填充颜色 | `string` | - |
