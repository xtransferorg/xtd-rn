# HightLightText 文本高亮

用于在文本中高亮显示指定的关键词。

## 何时使用

- 搜索结果中需要高亮显示关键词
- 文档或文章中需要突出显示重要内容
- 标签或分类中需要高亮匹配项
- 需要在长文本中快速定位特定内容
